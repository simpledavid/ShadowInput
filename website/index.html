ShadowInput

YouTube 字幕学习插件

下载插件 ZIP 查看安装步骤

核心功能

  • 鼠标悬停单词自动暂停视频，移开后自动恢复播放。
  • 英文字幕下方实时显示中文字幕，双语对照更直观。
  • 一键收藏生词，后续可导出复习。

安装方式（Chrome / Edge）

  1. 点击上方按钮下载 `shadowinput-extension.zip`。
  2. 解压 ZIP 到本地目录。
  3. 打开 `chrome://extensions`（Edge 打开 `edge://extensions`）。
  4. 开启右上角「开发者模式」，点击「加载已解压的扩展程序」，选择解压目录。
